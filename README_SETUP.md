# Настройка бота после обновления

## Новые функции

### 1. Обязательная подписка на канал
Теперь пользователи должны быть подписаны на ваш канал для использования бота.

### 2. Ограничение на 3 устройства
Каждый пользователь может подключить максимум 3 устройства одновременно.

## Настройка

### Шаг 1: Создайте канал в Telegram
1. Создайте публичный или приватный канал
2. Добавьте бота в канал как администратора
3. Дайте боту права на чтение списка участников

### Шаг 2: Получите ID канала

**Для публичного канала:**
- ID канала = `@your_channel_username`

**Для приватного канала:**
1. Перешлите любое сообщение из канала боту @userinfobot
2. Скопируйте ID (например: `-1001234567890`)

### Шаг 3: Обновите .env файл

Добавьте в файл `.env`:

```env
# ID канала (для публичного: @channel_name, для приватного: -1001234567890)
CHANNEL_ID=@your_channel_username

# Ссылка на канал для пользователей
CHANNEL_URL=https://t.me/your_channel

# Максимальное количество устройств (по умолчанию 3)
MAX_DEVICES=3
```

### Шаг 4: Перезапустите бота

```bash
python bot.py
```

## Как это работает

1. **Проверка подписки:**
   - При каждом действии бот проверяет подписку пользователя
   - Если не подписан - показывает кнопку "Подписаться на канал"
   - После подписки пользователь нажимает "Я подписался"
   - Админ не проходит проверку подписки

2. **Ограничение устройств:**
   - В 3X-UI панели параметр `limitIp` установлен в значение `MAX_DEVICES`
   - Пользователь может подключить только указанное количество устройств
   - При попытке подключить больше - старые соединения будут отключены

3. **Кнопка "Наш канал":**
   - Добавлена в главное меню
   - Ведет на ваш канал
   - Отображается только если `CHANNEL_URL` настроен

## Отключение проверки подписки

Если хотите временно отключить проверку подписки:
- Удалите или закомментируйте `CHANNEL_ID` в `.env`
- Перезапустите бота

## Изменение количества устройств

Чтобы изменить лимит устройств:
1. Измените `MAX_DEVICES` в `.env`
2. Перезапустите бота
3. Новые пользователи получат новый лимит
4. Для существующих пользователей лимит обновится при продлении подписки
