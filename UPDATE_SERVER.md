# Инструкция по обновлению бота на сервере

## Что было исправлено:

1. **services/vpn.py** - исправлен метод `create_client()`:
   - Теперь сначала получает текущие настройки inbound
   - Добавляет нового клиента к существующим
   - Добавлено подробное логирование всех шагов

2. **handlers/start.py** - улучшена логика реферальной системы:
   - Добавлены уведомления реферера о начислении бонуса
   - Показывается сколько дней осталось после начисления
   - Добавлено логирование всех операций

3. **.env** - исправлен формат `XRAY_PANEL_PATH` (убран слеш в начале)

## Как обновить на сервере:

### Вариант 1 - Через WinSCP (рекомендуется):

1. Подключись к серверу через WinSCP (IP: 213.176.65.246, user: root)
2. Перейди в папку `/root/vpn_bot/`
3. Замени файлы:
   - `services/vpn.py`
   - `handlers/start.py`
   - `.env`

### Вариант 2 - Через SSH:

```bash
# Подключись к серверу
ssh root@213.176.65.246

# Останови бота
supervisorctl stop vpn_bot

# Сделай бэкап старых файлов
cd /root/vpn_bot
cp services/vpn.py services/vpn.py.backup
cp handlers/start.py handlers/start.py.backup
cp .env .env.backup

# Теперь через WinSCP замени файлы или используй nano для редактирования

# Перезапусти бота
supervisorctl start vpn_bot

# Смотри логи
tail -f /var/log/vpn_bot.out.log
```

## Как протестировать:

1. Зарегистрируй нового пользователя через /start
2. В логах должны появиться строки:
   ```
   [Start] Создаём VPN клиента для user_id=...
   [XRay] Создаём клиента: user_id=...
   [XRay] Получаем inbound 1...
   [XRay] Отправляем запрос addClient...
   [XRay] Ответ addClient: {...}
   [XRay] Клиент успешно создан!
   ```

3. Проверь в боте "Мои ключи VPN" - должна появиться VLESS ссылка

4. Для теста реферальной системы:
   - Получи реферальную ссылку в боте
   - Зарегистрируй второго пользователя по этой ссылке
   - Первому пользователю должно прийти уведомление о начислении +3 дней

## Если что-то не работает:

Смотри логи в реальном времени:
```bash
tail -f /var/log/vpn_bot.out.log
```

Или последние 50 строк:
```bash
tail -n 50 /var/log/vpn_bot.out.log
```
